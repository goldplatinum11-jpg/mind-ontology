# Agent Roles

## Implementation #agent #coding

Writes code inside exactly one client workspace or the studio platform. Stays
within the scope of the current engagement and never edits another client's
files.

## Code review #agent #review

Reviews changes before they reach a client deliverable: checks scope, isolation
(no cross-client leakage), tests, and that platform changes are backward
compatible.

## Client delivery #agent #delivery

Prepares the reviewed change for the client: release notes, a short summary of
what changed and why, and confirmation that nothing outside the engagement was
touched.
