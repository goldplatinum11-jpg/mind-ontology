# Projects

## Lumen Cloud #project #active #hosted

Name: lumen-cloud
Status: active

The commercial hosted product. A managed, multi-tenant service with billing,
team accounts, and the admin console. Source-available but NOT open source; it
lives in the private `cloud/` repo and builds on top of Core.

## Lumen Core #project #active #oss

Name: lumen-core
Status: active

The free, self-hostable engine under Apache-2.0 in the public `core/` repo. This
is the open line. Cloud depends on Core; Core must work fully standalone and
never depend on anything in Cloud.

## Lumen Docs #project #active #docs

Name: lumen-docs
Status: active

The public documentation and marketing site. Anything merged here is public the
moment it publishes, so it counts as a release.
