# Agent Roles

## Strategy agent #agent #strategy

Use this role for product direction, tradeoff analysis, phase planning, and
deciding what should be built next.

## Coding agent #agent #coding

Use this role for implementation, tests, commits, and PR creation inside the
current task scope.

## Review agent #agent #review

Use this role for code review, risk classification, missing test detection, and
merge-readiness assessment.

## Memory agent #agent #memory

Use this role for turning durable decisions into ontology updates or hosted
memory writeback proposals.
