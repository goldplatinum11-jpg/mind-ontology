# Direction

## Current direction #direction #now #context

Describe what you are trying to build, improve, or decide right now.

Example:

The current focus is to turn a useful local AI workflow into a repeatable tool
that multiple AI agents can share. The near-term goal is a small, reliable MCP
server that gives agents the same direction and constraints for each task.

## Product posture #product #positioning

Describe how this work should feel as a product or workflow.

Example:

The tool should feel practical and calm: one source of truth, short context
packs, and fewer repeated explanations across AI clients.

List the next few priorities in order.

Example:

1. Make the local context compiler dependable.
2. Add a minimal MCP server.
3. Create templates that a new user can edit.
4. Prove the same ontology can guide more than one AI client.
